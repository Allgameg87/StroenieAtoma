Электронные оболочки

Электроны распределены по энергетическим уровням (орбитали), определяемым квантовыми числами:

  • n — главное квантовое число
  • l — азимутальное число
  • m — магнитное число

История моделей атома

Далтон (1803)

Атом как непроницаемый шарик. Объяснял химические реакции через соотношения масс.

Томсон (1904)

"Марципан с ягодами" — электронный пудинг. Электроны распределены в положительной среде.

Резерфорд (1911)

Ядро в центре, электроны вращаются вокруг. Объяснял эксперимент с альфа-частицами.

Бор (1913)

Квантовые орбиты. Электроны могут быть только на определенных орбитах.

Шредингер (1926)

Волновые функции. Электроны существуют в вероятностных областях (орбитали).

Квантовая механика

Ключевые принципы

  • Неопределенность Хайзенберга: Δx·Δp ≥ ħ/2
  • Суперпозиция: частица может находиться в нескольких состояниях одновременно
  • Квантовая энтангlement: коррелированные состояния частиц

Пример 1

Электрон в атоме водорода находится в состоянии 1s с энергией -13,6 эВ.

Пример 2

Спектр натрия: желтый свет (589 нм) возникает из перехода 3p → 2p.

Ядерная физика

Ядро атома

  • Ядерный радиус: R = R₀A¹/³ (R₀ ≈ 1,2 fm)
  • Ядерная сила: ультракороткодействующая, преодолевает электростатическое отталкивание
  • Ядерная энергия: ΔE = Δm·c² (формула Эйнштейна)

Радиоактивность

  • Альфа-распад: испускание α-частиц (₂He⁴)
  • Бета-распад: нейтрон → протон + электрон + нейтрино
  • Гамма-излучение: испускание γ-квантов

Периодическая система

Закономерности

  • Увеличение атомного номера Z → увеличение положительного заряда ядра
  • Схожесть свойств в группах (например, алькали-металлы)
  • Периодичность периодов: 2, 8, 18, 32 элементов

Электронные конфигурации

  • Приемник электронов: внешние оболочки (валентные электроны)
  • Закон Паули: заполнение орбиталей по 2 электронам
  • Энергетический порядок: 1s → 2s → 2p → 3s → 3p → 4s → 3d...

Практическое применение

Физика

  • Ядерные реакторы (цепная реакция)
  • Лазеры (стимулированное излучение)

Химия

  • Синтез новых материалов (например, полимеры)
  • Разработка лекарств (моделирование молекул)

Технологии

  • Квантовые компьютеры (квантовые биты)
  • Сцинтилляторы для медицинской томографии
